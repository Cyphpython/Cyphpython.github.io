Nier Hacktomata

The first Project of my second year at the HEAJ i made, this project is a 2D Bullet Hell Shooter made in SDL2 with C++. Our goal here was to make a game without a engine and make our own. It was a amazing way to learn new valuable concept

  • First Concept : Rendering (How to render something on the screen)
  • Second Concept : Entity (How to make a new entity without recopying the same code over and over )
  • Third Concept : Object Pooling (How to reuse a object isntead of recreating a new one)
  • Last Concept : Asset Manager (how to load a ressource and then use it)
